# Remove \[Square Brackets]
## SSMS
- Highlight script
	- Ctrl-H to get Quick Replace pop-up
		- Find... = `[\[\]]`
		- Alt-E to toggle on Use Regular Expressions (or click `.*` button)
		- Alt-A to Replace All (or click button)
